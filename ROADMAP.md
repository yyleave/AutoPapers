1. **高优先级**：Phase 1 数据底座（其他阶段依赖此基础）
2. **中优先级**：Phase 3 沙盒执行（核心工程难点）
3. **后续迭代**：Phase 2/4 可并行推进

## 风险点

| 风险 | 影响 | 缓解措施 |
|------|------|---------|
| API 限流 | 文献获取阻塞 | 异步队列 + 代理池 |
| PDF 解析质量 | 知识图谱噪声 | 多解析器交叉验证 |
| 代码执行安全 | 系统损坏 | 严格沙盒隔离 |
| 幻觉问题 | 论文可信度 | 强制 Grounding |
